windows环境下github+hexo建站

爱折腾
  • Hexo

自己搭建这个博客用了大概三四个小时（改了改主题什么的），查了很多篇文档，大致总结一下吧。掉坑的地方基本就是我用的是github桌面版，弄那个ssh的时候出了点问题。原来不用再搞ssh了（==）。

今天闲着无聊刷博客看到一篇文章，你为什要写博客，感觉是时候自己弄一个博客了。之前在csdn上写过几篇，感觉界面用着听不舒服的，于是就想自己搭建一个博客。其实我一开始想全部自己写的，但是凭借我加一起一共自学了两月的html+css+javascrapy，写了几百行代码我就已经崩溃了，还是老老实实的用框架吧QAQ。一共搞了一个下午大概3，4个小时吧。前几天买了一个域名，想和新浪云sae上我自己玩的网站绑定一下，但是按照sae的介绍进行并没有什么卵用，dns解析不了。刚好就给我正要建的博客用了。域名是lxjun.top(阿里云四软妹币买的==)。好把直接进入正题吧。网上查了一下博客的框架，较为收人推崇的是github+ hexo。github我是有的，没有的话直接申请就行了。

   github为用户提供了静态网页的代码空间。使用方法是：
    在github上新建一个名为 yourusername. github.io 的repository即可。然后先放着就行了

配置环境

   把node.js装好，傻瓜步骤，一直下一步就行了

   之后就是安装git，也是傻瓜步骤，一直下一步就行了

连接GitHub

本地Git连接GitHub需要配置密匙SSH，以便于本地Git与GitHub上的项目连接。在本地电脑上检查是否存在SSH：
$ cd ~/. ssh
检查本机的ssh密钥
如果提示：No such file or directory 说明你是第一次使用git
然后生成新的密匙：

$ ssh-keygen -t rsa -C "邮件地址@youremail.com"

然后系统会要你输入密码（这是设置密码）：

Enter passphrase (empty for no passphrase):<输入密码>

Enter same passphrase again:<再次输入加密串>

就成功设置ssh key了

打开本地C:\localhost.ssh\id_rsa.pub文件。此文件里面内容为刚才生成人密钥。如果看不到这个文件，你需要设置显示隐藏文件。准确的复制这个文件的内容，才能保证设置的成功。各电脑所在文件夹可能不同，在c盘搜索文件名就行了。

接下来在浏览器中打开你的github账号。点击右上角setting，进入SSH and GPG key项中，选择add ssh key。
把你本地生成的密钥复制到里面（key文本框中）， 点击 add key即可。

可以输入下面的命令，看看设置是否成功，git@github.com的部分不要修改：
$ ssh -T git@github.com
如果是下面的反馈：
The authenticity of host 'github.com (207.97.227.239)' can't be established. RSA key fingerprint is 16:27:ac:a5:76:28:2d:36:63:1b:56:4d:eb:df:a6:48. Are you sure you want to continue connecting (yes/no)?

输入yes看到：

Hi cnfeat! You've successfully authenticated, but GitHub does not provide shell access.

表示连接成功。

如果安装有github for windows桌面版的话，sshkey就不需要设置了，在桌面版点击右上角设置->options

将default shell选择为git bash即可。

安装hexo

Node和Git都安装好后,首先创建一个文件夹,如blog,用户存放hexo的配置文件,然后进入blog里安装Hexo。

右键代开git bash执行如下命令安装Hexo：

npm install-g hexo

然后，执行init命令初始化hexo,命令：

hexo init

hexo安装完毕，但是还并不会用，而且还没有和你的github关联

生成静态页面

hexo generate 可简写为 hexo g

启动本地服务，进行文章预览调试，命令：

hexo server 可简写为 hexo s

浏览器输入http://localhost:4000

到此hexo就安装完毕了，如果想更换主题的话可以参见hexo主题更换

用 hexo d将代码部署到你的github中即在浏览器中输入http://your_user_name.github.io来访问你的博客（your_user_name为你的github用户名）

绑定独立域名

自行申请独立域名，域名解析网站上添加A记录，在cmd中输入 ping http://your_user_name.github.io 来获取ip地址。地址为A记录的记录值，主机记录为@。再添加一个A记录，记录值还是这个ip，主机记录为www，然后在你的github 的your_user_name.github.io仓库中创建一个CNAME无类型文件，文件内容为你申请的域名（例：example.com）。等待几分钟即可通过独立域名访问你的博客了

不折腾版

在逛贴吧的时候发现了一个便携版，自称是免安装解压即用的Hexo便携版网站，Hexo便携版github。这个是面对完全小白的，按照提示一步步运行脚本就行了。
